# Volt

작사가를 위한 스마트 메모장 및 공개 영감 피드 프로젝트입니다.

## 실행 방법

```bash
npm install
npm run db:push
npm run dev
```

브라우저에서 `http://localhost:3000`으로 접속하면 됩니다.

## 주요 기능

- 공개 영감 피드
- 글귀 생성/수정/삭제
- 라임 메모 저장/삭제
- 개인 아카이브 필터링
- ko/en 다국어 지원
- 자체 로그인/회원가입
- 보호 라우트

## 검증 명령

```bash
npm run lint
npm run build
```
